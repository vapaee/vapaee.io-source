Order Book

Vapaée DEX is a decentralized exchange that allows you specify the price and amount of the token you want to buy or sell and create an order for the ledger.

The main currency is DEX and can be obtained by Staking TLOS in the Telos Wrapper

Swap Tokens

Vapaée DEX also offers instant token-swap convertion pools of token pairs, using DEX token as main currency.

The Swap is sync with its corresponding market's order book and it is smart enough to automatically take an order when the price is better than the swapping price.

Telos Wrapper

Vapaée DEX offers several ways for TLOS farming. You can use your TLOS as collateral to take a loan in DEX for the same amount, one DEX for each TLOS you deposit. But when you come back to cancel your loan, you will get back more than the initial amount of TLOS because we will put the in the REX system for you.

The DEX token is the main currency of the DEX.

Withdrawals

The old version of Vapaée DEX will be shut down on Q1 2022 and with that, all user orders will be canceled and the funds, along with the remaining deposits will be returned to the owners using an auto-withdrawal process.

join the telegram group to stay updated with the latest news about this auto-withdrawal.

Tokens contract

The vapaeetokens contract will drop all DEX functionality and keep only standard token functionality. That means all tokens will keep working as normal.

Later in the road map new great features will be added to the vapaeetokens contract and all tokens will be able to use them.
